# Texture Packing
Implémentations d'algorithmes de Texture Packing

## Liste des algorithmes implémentés :
* First Fit (FF)
* First Fit Decreasing Height (FFDH)
* Next Fit Decreasing Height (NFDH)
* Best Fit Decreasing Height (BFDH)
